Installation
============

Requirements
------------

+ Python 2.7 or Python 3.4 or later

  * http://www.python.org/
  * 2.7 and 3.7 are primary test environments.

+ pip and setuptools

  * http://pypi.org/project/pip/
  * http://pypi.org/project/setuptools/

Install
-------

Should be easy as ::

    python -m pip install --upgrade jtypes.jvm
