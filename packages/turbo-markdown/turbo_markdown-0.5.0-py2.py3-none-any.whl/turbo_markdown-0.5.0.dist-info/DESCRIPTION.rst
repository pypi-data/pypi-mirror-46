A cli tool to view local markdown files


* Free software: MIT license
* Documentation: https://turbo-markdown.readthedocs.io.


Get Started
-------------

Install and use turbo_markdown::

    pip install turbo-markdown
    cd markdown_dir/
    turbo-markdown
